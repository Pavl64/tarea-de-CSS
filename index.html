TEMAS IMPORTANTES DE CSS

-especificidad:

La especificidad en CSS es un mecanismo para controlar qué estilo se aplicará a un elemento cuando hay varias reglas CSS que se aplican al mismo elemento.

Tenemos cuatro niveles de epecificidad:

  • elementos HTML (especificidad=1)
  • clases (especificidad=10)
  • ID (especificidad=100)
  • inline (especificidad=1000)
  • important (especificidad=infinito)

    • ejemplo:

    h1{

    color:red;

    }


    .headline{

    color:firebrick

    }

    En el ejemplo anterior el color del h1 se establecera en firebrick, ya que el selector de clase es mas especifico que el selector de elemento


    -selectores:

    Los selectores en CSS son patrones que se utilizan para identificar los elementos HTML a los que se aplicarán los estilos. Los selectores pueden ser de diferentes tipos y se pueden combinar para crear selectores más complejos y específicos.

    Mostraremos algunos tipos de selectores en CSS:

    *selector de etiqueta: apunta a los elementos HTML especificos a los que se aplicara el estilo

    ejemplo:

    p{

    color:green;

    }

    En este ejemplo se observaa como cambiamos los estilos de los parrafos

    *selector de clase: es un atributo que va dentro de una etiqueta HTML

    ejemplo:

    (codigo HTML)

    < p class="brand">hola mundo < /p>


    (codigo css)

    .brand{

    color:teal;

    }

    *selector ID:es un atributo que va dentro de una etiueta HTML es el de mayor valor de especificidad

    ejemplo:

    (codigo HTML)

    < h1 id="largue-title"> aprendiendo estilos en css< /h1>


    (codigo css)

    #largue-title{

    font-family:cursive;

    }


    -cascada:

    la cascada en las hojas de estilo significa que el orden de las reglas importa en CSS: cuando dos reglas tienen la misma especificidad, se aplica la que aparece en último lugar en el CSS.

    ejemplo:

    h1{

    color:red;

    }

    h1{

    color:blue;

    }

    En este ejemplo se aplicara el color "blue" ya que tiene la misma especificidad y esta en ultimo lugar


    -herencia:

    Es el mecanismo mediante el cual determinadas propiedades de un elemento padre se transmiten a sus hijos.
